# Web UI 使用指南

  ## 访问方式

  1. **浏览器直接访问**：`http://localhost:8000/api/plugins/wb-manager/ui/`
  2. **通过扩展面板**：ST 扩展设置中点击 **📖 打开管理面板**
  3. **斜杠命令**：在 ST 聊天框中输入 `/wb-ui`

  ---

  ## 基本操作

  ### 浏览条目

  1. 从下拉菜单选择世界书
  2. 条目列表自动加载
  3. 支持按标题、关键字、内容实时搜索

  ### 编辑条目

  1. 点击条目右侧 **▼** 展开内联编辑器
  2. 修改激活策略、关键字、插入位置、正文等字段
  3. 点击 **✅ 应用** 保存到本地（内存中）
  4. 点击顶部 **💾 保存** 写入文件

  > 有未保存修改时，面板标题旁会显示橙色圆点 **●**

  ### 新建世界书

  点击世界书选择框右侧的 **＋** 按钮，弹出对话框后输入新世界书名称并确认。  
  创建成功后列表自动刷新并自动选中新世界书，同时后端创建空白 JSON 文件并同步到 ST 运行时。

  ### 新建条目

  点击面板底部 **＋ 新建条目**，展开编辑器，填写内容后点击 **✅ 应用**。

  ---

  ## 批量操作

  在条目列表中勾选多个条目后，底部批量操作区激活：

  - **激活策略** → 批量设置为常量/可选/向量化
  - **插入位置** → 批量设置 7 种位置之一
  - **Order/深度/概率** → 批量数值修改
  - **标题** → 批量重命名
  - **关键字** → 替换/添加/清空
  - **递归控制** → 批量设置防递归标志
  - **效果** → 批量设置粘性/冷却/延迟
  - **角色绑定** → 批量绑定角色过滤器
  - **批量启用/禁用** → 一键开关

  ---

  ## 双面板复制

  两个面板可以分别选择不同的世界书，然后：

  1. 在一个面板中勾选条目
  2. 点击 **→ 复制到右侧**（或 **← 复制到左侧**）
  3. 条目以新 UID 添加到目标面板
  4. 点击目标面板的 **💾 保存** 写入文件

  ---

  ## 导入世界书

  点击顶部 **📤 从文件** 按钮，选择本地 `.json` 格式的世界书文件导入。

  ---

  ## 激活策略说明

  | 图标 | 策略 | ST 字段 |
  |------|------|---------|
  | 🔵 | 常量（蓝灯） | `constant: true` |
  | 🟢 | 可选（绿灯） | `selective: true` |
  | 🔗 | 向量化 | `vectorized: true` |

  ---

  ## 插入位置简码

  | 简码 | 含义 |
  |------|------|
  | BC | 角色定义之前 |
  | AC | 角色定义之后 |
  | BE | 示例消息之前 |
  | AE | 示例消息之后 |
  | @D | 固定深度 |
  | BN | 作者注释之前 |
  | AN | 作者注释之后 |
